\doxysubsection{Detailed Description}
Entry point for the KL partition program. 

This program reads an adjacency matrix from a CSV file specified via command line or console input, performs KL partitioning with clustering, and writes results to output files\+:
\begin{DoxyItemize}
\item partition.\+txt \+: node -\/\texorpdfstring{$>$}{>} partition mapping
\item metrics.\+txt \+: cut and cluster variance metrics
\item trace.\+csv \+: cut evolution during KL swaps
